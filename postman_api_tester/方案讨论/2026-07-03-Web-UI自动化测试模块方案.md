## 1. 需求概述

在现有 Postman API 测试工具基础上，新增独立的 **Web UI 自动化测试模块**，核心能力：

1. **录制**：抓取用户在浏览器页面上的操作信息（点击、输入、导航、断言等）
2. **编辑**：提供 Web 端可视化编辑器，对录制的操作步骤进行增删改、添加断言
3. **执行**：将编辑后的用例以 UI 自动化脚本方式自动执行，生成测试报告

## 2. 行业方案调研

### 2.1 主流开源项目对比

| 项目 | Stars | 技术栈 | 录制方式 | 回放引擎 | 优势 | 劣势 |
|------|-------|--------|----------|----------|------|------|
| **Playwright Codegen** | ~87k | TypeScript/Node.js | CLI `codegen --ui`，浏览器内覆盖层实时捕获 | Playwright 原生 API（自动等待、智能选择器） | 代码质量最高；自动等待消除抖动；多浏览器支持；微软维护 | 需 Node.js 环境；非浏览器扩展形态 |

### 2.2 关键技术趋势（2025-2026）

1. **Record-to-Code 成为主流**：纯回放格式（如 Selenium IDE 内部格式）正在衰退，团队期望录制直接产出可维护代码
2. **CDP > WebDriver**：基于 Chrome DevTools Protocol 的工具（Playwright、Puppeteer）录制更快更可靠，直接拦截浏览器事件
3. **自动等待成为标配**：Playwright 引领的自动等待机制大幅减少因时序问题导致的测试抖动
4. **智能选择器**：从绝对 XPath 演进到语义化选择器（text、role、test-id），提高脚本健壮性

### 3.2 方案对比

| 维度 | 方案 A：Playwright Python 绑定 | 方案 B：Selenium WebDriver | 方案 C：CDP 直连 |
|------|-------------------------------|---------------------------|-----------------|
| **录制** | Chrome 扩展注入 JS，捕获操作后通过 WebSocket 发送到 Flask | 同左 | 同左 |
| **回放** | `playwright` Python 库（sync API） | `selenium` Python 库 | `pychrome` / `websocket-client` 直连 CDP |
| **选择器** | Playwright 智能选择器（text/role/test-id） | CSS/XPath | CSS/XPath |
| **自动等待** | 内置 | 需手动编写 WebDriverWait | 需手动实现 |
| **多浏览器** | Chromium + Firefox + WebKit | 全浏览器 | 仅 Chromium |
| **截图/视频** | 内置 | 需额外配置 | 需额外配置 |
| **与现有工具集成** | 高（Python 生态一致） | 高 | 中（需额外 CDP 封装） |
| **依赖复杂度** | `pip install playwright` + 浏览器二进制 | `pip install selenium` + WebDriver | 低但开发量大 |
| **推荐度** | ★★★★★ | ★★★ | ★★ |

**推荐方案 A**：Playwright Python 绑定

理由：
- 与现有 Python 技术栈完全一致（Flask + requests 生态）
- 自动等待 + 智能选择器大幅降低脚本抖动
- 内置截图/视频/tracing，报告内容丰富
- 一套脚本跑 Chromium + Firefox + WebKit
- `playwright` Python 库 API 成熟，社区活跃

### 3.3 核心数据模型

```python
class UIStep(TypedDict):
    """单个操作步骤"""
    index: int
    action: str           # click | type | select | navigate | wait | assert
    selector: str         # Playwright 选择器（text/role/css/xpath）
    selector_type: str    # text | role | css | xpath | test-id
    value: str            # 输入值/选择值/断言期望值
    description: str      # 步骤描述（可选）
    screenshot: bool      # 是否在此步骤截图
    timeout_ms: int       # 步骤超时（默认 30000）

class UIAssertion(TypedDict):
    """断言配置"""
    type: str             # visible | text | url | title | element_count
    selector: str
    expected: str
    negate: bool          # 反向断言

class UITestCase(TypedDict):
    """UI 自动化测试用例"""
    id: str
    name: str
    folder: str
    base_url: str
    steps: List[UIStep]
    assertions: List[UIAssertion]
    variables: Dict[str, str]  # 参数化变量
    tags: List[str]
    created_at: str
    updated_at: str

class UITestSuite(TypedDict):
    """UI 自动化测试套件"""
    name: str
    cases: List[UITestCase]
    global_variables: Dict[str, str]
    browser: str          # chromium | firefox | webkit
    headless: bool
    viewport: Dict[str, int]  # {width, height}
```

#### 方案 A：Automa（现有扩展，零开发）

**使用方式**：
1. Chrome Web Store 安装 Automa（免费）
2. 创建新 Workflow，添加 "New Tab" + "Recorder" 节点
3. 录制页面操作，自动生成节点流
4. 添加 "HTTP Request" 节点，POST 到 Flask API `/api/ui-testing/import-automa`
5. Flask 解析 Automa JSON，转换为内部 UITestCase 格式

**Automa JSON 结构示例**：
```json
{
  "nodes": [
    {"id": "1", "type": "newtab", "data": {"url": "https://example.com"}},
    {"id": "2", "type": "click", "data": {"selector": "#login-btn", "markEl": true}},
    {"id": "3", "type": "forms", "data": {"selector": "#username", "type": "text-field", "value": "admin"}},
    {"id": "4", "type": "forms", "data": {"selector": "#password", "type": "text-field", "value": "123456"}},
    {"id": "5", "type": "click", "data": {"selector": "button[type=submit]"}}
  ],
  "edges": [{"from": "1", "to": "2"}, {"from": "2", "to": "3"}]
}
```

**Flask 转换逻辑**：
```python
def convert_automa_to_steps(automa_json: dict) -> List[UIStep]:
    steps = []
    for i, node in enumerate(automa_json.get("nodes", [])):
        node_type = node.get("type")
        data = node.get("data", {})
        if node_type == "click":
            steps.append({"action": "click", "selector": data["selector"], ...})
        elif node_type == "forms":
            steps.append({"action": "type", "selector": data["selector"], "value": data["value"], ...})
        elif node_type == "newtab":
            steps.append({"action": "navigate", "value": data["url"], ...})
    return steps
```

##### B.4 事件捕获详细设计

**recorder.js 核心逻辑**：

```javascript
class UIRecorder {
  constructor() {
    this.recording = false;
    this.sessionId = null;
    this.lastInputTarget = null;
    this.inputBuffer = { element: null, value: '', timer: null };
    this.pendingNavigation = null;
    this.boundHandlers = {};
  }

  start(sessionId) {
    this.sessionId = sessionId;
    this.recording = true;

    // 绑定事件监听（capture 阶段，确保先于页面脚本）
    const events = [
      ['click', this.handleClick, true],
      ['dblclick', this.handleDblClick, true],
      ['input', this.handleInput, true],
      ['change', this.handleChange, true],
      ['submit', this.handleSubmit, true],
      ['keydown', this.handleKeydown, true],
      ['scroll', this.handleScroll, false],   // bubble 阶段
    ];
    events.forEach(([evt, fn, capture]) => {
      const bound = fn.bind(this);
      document.addEventListener(evt, bound, capture);
      this.boundHandlers[evt] = bound;
    });

    // 监听页面导航
    window.addEventListener('popstate', this.handlePopState.bind(this));
    window.addEventListener('hashchange', this.handleHashChange.bind(this));

    // 监听链接点击（捕获跳转前的 href）
    document.addEventListener('click', this.handleLinkClick.bind(this), true);
  }

  // ── 点击事件 ──
  handleClick(event) {
    if (!this.recording) return;
    const el = event.target;

    // 忽略录制器自身的 UI 元素
    if (el.closest('[data-recorder-ignore]')) return;

    // 如果点击的是链接，标记为"可能触发导航"
    const link = el.closest('a[href]');
    if (link) {
      this.pendingNavigation = {
        trigger: 'click_link',
        href: link.href,
        timestamp: Date.now()
      };
    }

    this.sendStep({
      action: 'click',
      selector: SelectorEngine.generate(el),
      value: '',
      element_info: this.getElementInfo(el),
      page_url: location.href,
      page_title: document.title,
      coordinates: { x: event.clientX, y: event.clientY }
    });
  }

  // ── 输入事件（防抖合并连续输入） ──
  handleInput(event) {
    if (!this.recording) return;
    const el = event.target;
    if (!el.matches('input, textarea, [contenteditable]')) return;

    // 防抖：连续输入合并为一次 type 操作
    clearTimeout(this.inputBuffer.timer);
    this.inputBuffer.element = el;
    this.inputBuffer.value = el.value || el.textContent;
    this.inputBuffer.timer = setTimeout(() => {
      this.sendStep({
        action: 'type',
        selector: SelectorEngine.generate(this.inputBuffer.element),
        value: this.inputBuffer.value,
        element_info: this.getElementInfo(this.inputBuffer.element),
        page_url: location.href,
        page_title: document.title,
        input_type: this.inputBuffer.element.type || 'text'
      });
      this.inputBuffer = { element: null, value: '', timer: null };
    }, 500);  // 500ms 内无新输入则提交
  }

  // ── 下拉选择 ──
  handleChange(event) {
    if (!this.recording) return;
    const el = event.target;
    if (el.tagName === 'SELECT') {
      this.sendStep({
        action: 'select',
        selector: SelectorEngine.generate(el),
        value: el.value,
        element_info: { ...this.getElementInfo(el), selected_text: el.options[el.selectedIndex]?.text },
        page_url: location.href,
        page_title: document.title
      });
    }
    // checkbox / radio
    if (el.type === 'checkbox' || el.type === 'radio') {
      this.sendStep({
        action: el.checked ? 'check' : 'uncheck',
        selector: SelectorEngine.generate(el),
        value: String(el.checked),
        element_info: this.getElementInfo(el),
        page_url: location.href,
        page_title: document.title
      });
    }
  }

  // ── 表单提交 ──
  handleSubmit(event) {
    if (!this.recording) return;
    this.sendStep({
      action: 'submit',
      selector: SelectorEngine.generate(event.target),
      value: '',
      element_info: this.getElementInfo(event.target),
      page_url: location.href,
      page_title: document.title
    });
  }

  // ── 键盘事件（仅捕获特殊键） ──
  handleKeydown(event) {
    if (!this.recording) return;
    const specialKeys = ['Enter', 'Tab', 'Escape', 'ArrowUp', 'ArrowDown'];
    if (!specialKeys.includes(event.key)) return;
    // 如果焦点在输入框且是 Enter，等 submit 事件处理
    if (event.key === 'Enter' && event.target.matches('input')) return;

    this.sendStep({
      action: 'keypress',
      selector: SelectorEngine.generate(event.target),
      value: event.key,
      element_info: this.getElementInfo(event.target),
      page_url: location.href,
      page_title: document.title,
      modifiers: {
        ctrl: event.ctrlKey, shift: event.shiftKey,
        alt: event.altKey, meta: event.metaKey
      }
    });
  }

  // ── 页面导航 ──
  handleLinkClick(event) {
    if (!this.recording) return;
    const link = event.target.closest('a[href]');
    if (!link) return;

    // 检测是否新开标签/窗口
    const isNewTab = link.target === '_blank' || event.ctrlKey || event.metaKey;

    // 延迟发送导航消息（等页面实际跳转）
    setTimeout(() => {
      if (location.href !== link.href || isNewTab) {
        this.sendNavigation({
          action: 'navigate',
          trigger: 'click_link',
          from_url: this._lastUrl || location.href,
          to_url: link.href,
          is_new_tab: isNewTab,
          is_new_window: false,
          tab_id: null  // background.js 填充
        });
      }
    }, 100);
  }

  handlePopState() {
    if (!this.recording) return;
    this.sendNavigation({
      action: 'navigate',
      trigger: 'history_back_forward',
      from_url: this._lastUrl,
      to_url: location.href,
      is_new_tab: false,
      is_new_window: false
    });
    this._lastUrl = location.href;
  }

  // ── 发送消息到 background.js ──
  sendStep(data) {
    chrome.runtime.sendMessage({
      type: 'step',
      session_id: this.sessionId,
      timestamp: Date.now(),
      data: data
    });
  }

  sendNavigation(data) {
    chrome.runtime.sendMessage({
      type: 'navigation',
      session_id: this.sessionId,
      timestamp: Date.now(),
      data: data
    });
  }

  getElementInfo(el) {
    return {
      tag: el.tagName.toLowerCase(),
      text: (el.textContent || '').trim().substring(0, 100),
      type: el.type || '',
      href: el.href || '',
      name: el.name || '',
      placeholder: el.placeholder || '',
      visible: el.offsetParent !== null,
      aria_label: el.getAttribute('aria-label') || '',
      test_id: el.getAttribute('data-testid') || ''
    };
  }
}
```

##### B.5 选择器生成引擎

```javascript
class SelectorEngine {
  static generate(el) {
    const strategies = [
      () => this.byTestId(el),
      () => this.byAriaRole(el),
      () => this.byText(el),
      () => this.byId(el),
      () => this.byNameAttr(el),
      () => this.byCssShort(el),
      () => this.byXPath(el),
    ];

    for (const strategy of strategies) {
      const result = strategy();
      if (result) return {
        primary: result.selector,
        strategy: result.strategy,
        fallback_css: this.byId(el)?.selector || this.byCssShort(el)?.selector || '',
        fallback_xpath: this.byXPath(el)?.selector || ''
      };
    }
    return { primary: this.byXPath(el).selector, strategy: 'xpath', fallback_css: '', fallback_xpath: '' };
  }

  static byTestId(el) {
    const id = el.getAttribute('data-testid') || el.getAttribute('data-test') || el.getAttribute('data-cy');
    if (id) return { selector: `[data-testid="${id}"]`, strategy: 'test-id' };
    return null;
  }

  static byAriaRole(el) {
    const role = el.getAttribute('role') || this._inferRole(el);
    const name = el.getAttribute('aria-label') || el.textContent?.trim().substring(0, 50);
    if (role && name) return { selector: `role=${role}[name="${name}"]`, strategy: 'role' };
    return null;
  }

  static byText(el) {
    const text = el.textContent?.trim();
    if (text && text.length <= 50 && !el.children.length) {
      return { selector: `text="${text}"`, strategy: 'text' };
    }
    return null;
  }

  static byId(el) {
    if (el.id && !el.id.match(/^\d/) && !el.id.includes(':')) {
      return { selector: `#${CSS.escape(el.id)}`, strategy: 'id' };
    }
    return null;
  }

  static byNameAttr(el) {
    const name = el.getAttribute('name');
    if (name) return { selector: `[name="${name}"]`, strategy: 'name' };
    return null;
  }

  static byCssShort(el) {
    // 生成最短唯一 CSS 路径
    const path = [];
    let current = el;
    while (current && current !== document.body) {
      let segment = current.tagName.toLowerCase();
      if (current.id) { segment = `#${CSS.escape(current.id)}`; path.unshift(segment); break; }
      if (current.className) {
        const classes = Array.from(current.classList).slice(0, 2).map(c => `.${CSS.escape(c)}`).join('');
        segment += classes;
      }
      const parent = current.parentElement;
      if (parent) {
        const siblings = Array.from(parent.children).filter(c => c.tagName === current.tagName);
        if (siblings.length > 1) {
          const index = siblings.indexOf(current) + 1;
          segment += `:nth-of-type(${index})`;
        }
      }
      path.unshift(segment);
      current = current.parentElement;
      // 检查当前路径是否已唯一
      const candidate = path.join(' > ');
      if (document.querySelectorAll(candidate).length === 1) break;
    }
    return { selector: path.join(' > '), strategy: 'css' };
  }

  static byXPath(el) {
    const parts = [];
    let current = el;
    while (current && current.nodeType === Node.ELEMENT_NODE) {
      let index = 0;
      let sibling = current.previousSibling;
      while (sibling) {
        if (sibling.nodeType === Node.ELEMENT_NODE && sibling.tagName === current.tagName) index++;
        sibling = sibling.previousSibling;
      }
      const tagName = current.tagName.toLowerCase();
      const part = index > 0 ? `${tagName}[${index + 1}]` : tagName;
      parts.unshift(part);
      current = current.parentNode;
    }
    return { selector: '/' + parts.join('/'), strategy: 'xpath' };
  }

  static _inferRole(el) {
    const tag = el.tagName.toLowerCase();
    const type = el.type;
    if (tag === 'button') return 'button';
    if (tag === 'a') return 'link';
    if (tag === 'input' && type === 'text') return 'textbox';
    if (tag === 'input' && type === 'checkbox') return 'checkbox';
    if (tag === 'input' && type === 'radio') return 'radio';
    if (tag === 'select') return 'combobox';
    if (tag === 'textarea') return 'textbox';
    if (tag === 'img') return 'img';
    if (tag === 'nav') return 'navigation';
    if (tag === 'main') return 'main';
    return null;
  }
}
```

```javascript
// 监听来自 content script 的消息

    ws.send(JSON.stringify(msg));
  }
});

function startRecording(wsUrl, caseName) {
  ws = new WebSocket(wsUrl);
  ws.onopen = () => {
    ws.send(JSON.stringify({
      type: 'session_start',
      case_name: caseName,
      timestamp: Date.now()
    }));
  };
  ws.onmessage = (event) => {
    const msg = JSON.parse(event.data);
    if (msg.type === 'session_ready') {
      sessionId = msg.session_id;
      // 通知所有 content script 开始录制
      chrome.tabs.query({}, (tabs) => {
        tabs.forEach(tab => {
          chrome.tabs.sendMessage(tab.id, {
            type: 'start',
            session_id: sessionId
          }).catch(() => {});  // 忽略无 content script 的标签
        });
      });
    }
  };
  ws.onclose = () => { stopRecording(); };
}

function stopRecording() {
  if (ws?.readyState === WebSocket.OPEN) {
    ws.send(JSON.stringify({ type: 'session_end', session_id: sessionId, timestamp: Date.now() }));
    ws.close();
  }
  ws = null;
  sessionId = null;
  // 通知所有 content script 停止录制
  chrome.tabs.query({}, (tabs) => {
    tabs.forEach(tab => {
      chrome.tabs.sendMessage(tab.id, { type: 'stop' }).catch(() => {});
    });
  });
}
```

##### B.7 Flask WebSocket 端点（接收录制数据）

```python
# postman_api_tester/handlers/ui_recorder_routes.py

from flask import request
from flask_socketio import emit, Namespace
import uuid
import json
import logging

logger = logging.getLogger(__name__)

# 录制会话存储（内存，生产可换 Redis）
_recording_sessions: dict = {}


class UIRecorderNamespace(Namespace):
    """WebSocket 命名空间：/ws/ui-recorder"""

    def on_connect(self):
        logger.info("UI recorder client connected: %s", request.sid)

    def on_disconnect(self):
        # 查找并清理该连接的会话
        for sid, session in list(_recording_sessions.items()):
            if session.get("flask_sid") == request.sid:
                session["status"] = "disconnected"

    def on_session_start(self, data):
        session_id = uuid.uuid4().hex[:12]
        case_name = data.get("case_name", "未命名用例")
        _recording_sessions[session_id] = {
            "flask_sid": request.sid,
            "case_name": case_name,
            "steps": [],
            "navigations": [],
            "status": "recording",
            "created_at": data.get("timestamp"),
        }
        emit("session_ready", {"session_id": session_id})
        logger.info("Recording session started: %s (%s)", session_id, case_name)

    def on_step(self, data):
        session_id = data.get("session_id")
        session = _recording_sessions.get(session_id)
        if not session:
            return
        step = data.get("data", {})
        step["index"] = len(session["steps"])
        session["steps"].append(step)
        # 实时回传确认 + 步骤摘要（供前端更新 UI）
        emit("step_ack", {
            "step_index": step["index"],
            "action": step.get("action"),
            "selector_primary": step.get("selector", {}).get("primary", ""),
            "value_preview": (step.get("value") or "")[:50],
        })

    def on_navigation(self, data):
        session_id = data.get("session_id")
        session = _recording_sessions.get(session_id)
        if not session:
            return
        nav = data.get("data", {})
        nav["index"] = len(session["navigations"])
        session["navigations"].append(nav)
        emit("navigation_ack", {
            "index": nav["index"],
            "to_url": nav.get("to_url", ""),
            "is_new_tab": nav.get("is_new_tab", False),
        })

    def on_session_end(self, data):
        session_id = data.get("session_id")
        session = _recording_sessions.get(session_id)
        if not session:
            return
        session["status"] = "completed"
        # 合并 steps + navigations 为完整步骤序列
        merged = self._merge_steps_and_navigations(session)
        # 保存为 UITestCase JSON
        case = self._build_test_case(session, merged)
        # 持久化（复用现有文件存储模式）
        from postman_api_tester.services.ui_testing_service import save_ui_case
        save_ui_case(case)
        emit("session_complete", {
            "session_id": session_id,
            "case_id": case["id"],
            "step_count": len(merged),
        })
        logger.info("Recording session completed: %s (%d steps)", session_id, len(merged))

    def _merge_steps_and_navigations(self, session):
        """按时间戳合并 steps 和 navigations 为有序步骤列表。"""
        all_events = []
        for s in session["steps"]:
            all_events.append({**s, "_source": "step"})
        for n in session["navigations"]:
            all_events.append({**n, "_source": "navigation"})
        all_events.sort(key=lambda e: e.get("timestamp", 0))
        # 重新编号
        for i, evt in enumerate(all_events):
            evt["index"] = i
        return all_events

    def _build_test_case(self, session, merged_steps):
        """构建 UITestCase 数据结构。"""
        return {
            "id": uuid.uuid4().hex[:12],
            "name": session["case_name"],
            "folder": "录制用例",
            "base_url": merged_steps[0].get("page_url", "") if merged_steps else "",
            "steps": [
                {
                    "index": s["index"],
                    "action": s.get("action", ""),
                    "selector": s.get("selector", {}).get("primary", "") if isinstance(s.get("selector"), dict) else s.get("selector", ""),
                    "selector_fallbacks": {
                        "css": s.get("selector", {}).get("fallback_css", "") if isinstance(s.get("selector"), dict) else "",
                        "xpath": s.get("selector", {}).get("fallback_xpath", "") if isinstance(s.get("selector"), dict) else "",
                    },
                    "value": s.get("value", ""),
                    "description": self._auto_describe(s),
                    "page_url": s.get("page_url", s.get("to_url", "")),
                    "screenshot": False,
                    "timeout_ms": 30000,
                }
                for s in merged_steps
            ],
            "assertions": [],
            "variables": {},
            "tags": ["recorded"],
            "created_at": session.get("created_at"),
        }

    def _auto_describe(self, step):
        """自动生成步骤描述文本。"""
        action = step.get("action", "")
        info = step.get("element_info", {})
        text = info.get("text", "")[:30]
        tag = info.get("tag", "")
        url = step.get("to_url", step.get("page_url", ""))

        if action == "click":
            return f"点击 {tag}「{text}」" if text else f"点击 {tag}"
        elif action == "type":
            return f"输入「{step.get('value', '')[:20]}」到 {info.get('placeholder', tag)}"
        elif action == "select":
            return f"选择「{info.get('selected_text', step.get('value', ''))}」"
        elif action in ("check", "uncheck"):
            return f"{'勾选' if action == 'check' else '取消勾选'} {text or tag}"
        elif action == "navigate":
            return f"导航到 {url}"
        elif action == "new_tab":
            return f"新建标签页 → {url}"
        elif action == "keypress":
            return f"按下 {step.get('value', '')} 键"
        elif action == "submit":
            return f"提交表单"
        return action
```

```python
# 伪代码：UI 自动化执行器
class UITestExecutor:
    def __init__(self, browser: str = "chromium", headless: bool = True):
        self.playwright = sync_playwright().start()
        self.browser = getattr(self.playwright, browser).launch(headless=headless)

    def execute_case(self, case: UITestCase) -> UITestResult:
        context = self.browser.new_context(viewport=case.get("viewport"))
        page = context.new_page()
        results = []

        for step in case["steps"]:
            try:
                self._execute_step(page, step, case.get("variables", {}))
                results.append({"step": step, "status": "PASSED", "screenshot": self._take_screenshot(page, step)})
            except Exception as e:
                results.append({"step": step, "status": "FAILED", "error": str(e), "screenshot": self._take_screenshot(page, step)})
                break

        context.close()
        return {"case": case, "steps": results, "status": "PASSED" if all(r["status"] == "PASSED" for r in results) else "FAILED"}

    def _execute_step(self, page, step, variables):
        action = step["action"]
        selector = substitute_variables(step["selector"], variables)
        value = substitute_variables(step.get("value", ""), variables)

        if action == "click":
            page.click(selector, timeout=step.get("timeout_ms", 30000))
        elif action == "type":
            page.fill(selector, value)
        elif action == "navigate":
            page.goto(value)
        elif action == "wait":
            page.wait_for_selector(selector)
        elif action == "assert":
            self._run_assertion(page, step)
```

3. **执行引擎是否需要支持并发？** — Phase 1 串行执行，Phase 4 可复用现有并发基础设施
4. **用例存储格式？** — 建议 JSON 文件（与现有 Collection 格式一致），后续可选 SQLite
