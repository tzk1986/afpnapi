## ⚠️ 遇到的问题汇总

### 问题1：Seldom TestCase 属性冲突 ✅ 已解决

**发现时间**: 第一次集成测试时  
**错误信息**:
```
AttributeError: can't set attribute 'response'
AttributeError: can't set attribute 'status_code'
```

**根本原因**:  
`PostmanTestExecutor` 继承 `seldom.TestCase` 时，`response` 和 `status_code` 被 TestCase 占用为只读属性。

**解决方案**:
```python
# ❌ 问题代码
class PostmanTestExecutor(seldom.TestCase):
    def __init__(self, api_config: Dict):
        super().__init__()
        self.response = None          # 冲突
        self.status_code = None       # 冲突

# ✅ 修复代码
class PostmanTestExecutor:
    def __init__(self, api_config: Dict):
        import requests
        self.http_response = None     # 改名
        self.resp_status_code = None  # 改名
        self.session = requests.Session()  # 直接使用 requests
```

**验证结果**: ✅ 测试成功通过

---

### 问题2：SSL 连接超时 ⚠️ 网络问题

**发现时间**: PUT 请求执行时  
**错误信息**:
```
SSLError(SSLEOFError(8, 'EOF occurred in violation of protocol (_ssl.c:1007)'))
```

**原因分析**:  
- 网络连接不稳定
- SSL/TLS 握手被中断
- 与测试服务器相关

**建议改进**:
```python
# 可选的改进方案
import ssl
requests.packages.urllib3.disable_warnings()

session = requests.Session()
session.verify = False  # 仅用于测试环境

# 或添加重试机制
from urllib3.util.retry import Retry
from requests.adapters import HTTPAdapter

retry = Retry(
    total=3,
    backoff_factor=0.5,
    status_forcelist=[500, 502, 503, 504]
)
adapter = HTTPAdapter(max_retries=retry)
session.mount('https://', adapter)
```

**当前处理**: ✅ 1 个 API 错误可接受（网络问题），其余 4 个通过
